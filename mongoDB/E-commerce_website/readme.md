# E-Commerce Website - Full-Stack Node.js + Express + MongoDB

A fully functional **e-commerce backend API** built with the MERN stack (MongoDB, Express.js, Node.js).

## Features

- RESTful API with clean architecture
- Full CRUD for products
- Environment-based configuration
- Tested with Thunder Client (collection included)

<br/>

**Technologies Used:**  
Node.js • Express.js • MongoDB • Mongoose • Thunder Client (for testing)

## Quick Start

```bash
git clone https://github.com/tanvii-18/E-commerce_website.git
cd E-commerce_website
npm install
npm start
```
