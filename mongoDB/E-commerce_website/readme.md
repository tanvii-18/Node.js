# E-Commerce Website - Full-Stack Node.js + Express + MongoDB

A fully functional **e-commerce backend API** built with the MERN stack (MongoDB, Express.js, Node.js). Includes user authentication, product management, cart, orders, and admin features.

## Features

- RESTful API with clean architecture
- JWT-based authentication & role-based access (user/admin)
- Full CRUD for products, cart & orders
- Secure password hashing with bcrypt
- Environment-based configuration
- Tested with Thunder Client (collection included)
